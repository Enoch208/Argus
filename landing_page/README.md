# Argus Landing Page

Marketing site for Argus: a private AI security layer for Solana signatures.

The landing page is designed to communicate three ideas quickly:

- Argus reviews transactions before users sign.
- The AI review runs locally.
- Verdicts are grounded in decode, simulation, OCR, threat intel, and personal history.

## Stack

- Next.js
- React
- Tailwind
- Shared visual language with the desktop app

## Development

```bash
npm install
npm run dev
```

Open the local URL printed by Next.js.

## Positioning Notes

The page should feel like infrastructure-grade security, not a generic AI assistant:

- Lead with privacy and self-custody.
- Show concrete review signals.
- Avoid vague claims like “AI-powered protection” without explaining the pipeline.
- Keep the tone premium, technical, and direct.
